# 🛒 MyStore - PHP E-commerce Web Application

MyStore is a simple e-commerce web application built with PHP and MySQL. It allows users to register, log in, browse products, add items to a shopping cart, and complete orders.

The project focuses on core backend functionality such as user authentication, session handling, and cart management.

---

## 🚀 Features

- User registration (Sign Up)
- User authentication (Login / Logout)
- Product listing system
- Add items to shopping cart
- Remove items from cart
- Cart total calculation
- Order confirmation (Success page)
- Session-based user management
- Bootstrap-based responsive UI

---

## 🔐 Authentication

- **Signup:** New users can create an account
- **Login:** Existing users can access their account
- **Logout:** Secure session termination
- **Sessions:** Used to track logged-in users

---

## 🧾 Application Flow

1. User registers an account
2. User logs in
3. Products are displayed
4. User adds products to cart
5. Cart page displays selected items and total price
6. Order is confirmed via success page
7. User logs out
